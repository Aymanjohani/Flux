# Test Knowledge Base

## Project Overview
This is a test document for the Flux-Lite memory system. It demonstrates how the system ingests and retrieves information from markdown files.

## Technology Stack
The Flux-Lite system uses:
- LanceDB for vector storage
- Gemini API for embeddings
- Node.js for the runtime
- Markdown for knowledge representation

## Key Features
- Lightweight and serverless
- File-based architecture
- Semantic search capabilities
- Automatic chunking by headers
